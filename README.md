# 工具

[day.js](https://github.com/iamkun/dayjs)

一个 2KB 的 JS 日期时间库，和 Moment.js 的 API 设计保持完全一样。

[iconsvg](<https://iconsvg.xyz/>)

一套可以定制的 SVG 图标。

[gitmoji-cli](<https://github.com/carloscuesta/gitmoji-cli>)

在 git commit message 里面使用 emoji 的工具。
